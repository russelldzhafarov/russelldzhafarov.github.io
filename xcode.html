Sympli Plugin for Xcode

Sympli plugin for Xcode helps developers implement designs made with Sketch, Figma, Adobe XD or Photoshop.

Using Sympli plugin for Xcode you can click through every single element of the design and get detailed information, including dimensions, position, effects, fonts used, text size, alignment, etc.

With Sympli plugin for Xcode you can apply properties from design to views in the Interface Builder. This works for positions, dimensions, images, and text labels. You can also use the Sympli Handoff plugin with your Xcode Simulator.
